# English Adaptive Quiz

Single-file adaptive quiz app for EPG 600 / EnglishScore Core Skills practice.

## Scope

- Grammar & Vocab
- Reading
- Listening script practice
- Adaptive difficulty
- Weakness map and review queue

## Run

Open `index.html` directly in a browser, or serve this folder with any static file server.

## Assets

- `assets/icon.png` app icon
- `assets/hero.jpg` hero banner
